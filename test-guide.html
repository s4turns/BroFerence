üêõ Troubleshooting

No remote video?

  • Check both windows are in the same room - Room name must match exactly
  • Press F12 in browser ‚Üí Console tab ‚Üí Look for errors
  • Check "Users in Room" appears in both windows
  • Did you click "Call"? The connection doesn't start automatically

Can't join room?

  • Check status shows "Connected" before joining room
  • Refresh the page and try again
  • Check signaling server is running: docker compose ps

Camera/mic not working?

  • Check browser permissions (click lock icon in address bar)
  • Try a different browser (Chrome/Edge recommended)
  • Make sure camera/mic aren't being used by another app
